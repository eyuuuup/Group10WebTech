API documentation
Resource Request verb Description Status Code
/api/products GET Retrieve the full data set of products 200 OK
/api/products POST Add data for a new product item using JSON 201 CREATED
/api/products/:id GET Retrieve data of a specific item 200 OK
/api/products/:id PUT Change data of a specific item 204 No Content
/api/products/:id DELETE Remove data of a specific item 204 No Content
/api/products/reset DELETE Reset the database to an empty state 204 No Content

GET /api/products/:id

Retrieve data for a specific item, response will be formatted in JSON. Requires an ID as parameter.

Example Request

GET http://localhost:3000/api/products/23

Example Response

    {
        "id": 23,
        "brand": "Samsung",
        "model": "Galaxy S7",
        "os": "Andriod",
        "image": "https://s.s-bol.com/imgbase0/imagebase3/large/FC/6/4/2/1/9200000054961246.jpg",
        "screensize": 5
    }

POST /api/products

Add new data to the database, request must be formatted using JSON. ID can be given as a parameter, but isn't needed.

Example Request

POST http://localhost:3000/api/products

Example Response

        {
            "message":"success",
            "data":
            {
                "ID":"",
                "brand":"iPhone",
                "model":"iPhone XS",
                "os":"iOS",
                "image":"https://image.coolblue.nl/max/500x500/products/1121127",
                "screensize":"5"
            },
            "id":27
        }

PUT /api/products/:id

Updates existing product in database with ID given. Request must be given as JSON.

Example Request

PUT http://localhost:3000/api/products/:id

Example Response

Only sends back 204 status code.

DELETE /api/products/:id

Deletes specific data entry in the database.

Example Request

DELETE http://localhost:3000/api/products/:id

Example Response

Only sends back 204 status code.

DELETE /api/products/reset

Resets the whole database.

Example Request

DELETE http://localhost:3000/api/products/reset

Example Response

Only sends back 204 status code.
